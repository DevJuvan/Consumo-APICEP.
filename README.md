# 📍 Cadastro de Endereço com API ViaCEP

Projeto desenvolvido com HTML, CSS e JavaScript que consome a API ViaCEP para preencher automaticamente os campos de endereço a partir do CEP informado pelo usuário.

---

## 🚀 Funcionalidades

- ✔ Busca automática de endereço pelo CEP
- ✔ Preenchimento automático de:
  - Rua
  - Bairro
  - Cidade
  - Estado
- ✔ Validação básica de CEP
- ✔ Tratamento de erro para CEP inválido

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- API ViaCEP

---

## 📡 Como Funciona

1. O usuário digita um CEP.
2. O JavaScript faz uma requisição para a API ViaCEP.
3. A API retorna os dados em formato JSON.
4. Os campos do formulário são preenchidos automaticamente.

Exemplo de requisição:
https://viacep.com.br/ws/01001000/json/
